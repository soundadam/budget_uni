# Notebooks

后续分析可以放在这里。优先建议用带 `# %%` 单元的 interactive Python 脚本，例如：

```text
notebooks/01_budget_overview.py
notebooks/02_rank_changes.py
```

分析脚本默认读取：

```text
../data/processed/ministry_university_budget.csv
```

常见分析方向：

1. 每年教育部直属高校预算总量。
2. 各高校年度排名变化。
3. 各高校预算同比增速。
4. 顶部高校和中位数高校的差距变化。
5. 口径异常或缺失数据标注。
